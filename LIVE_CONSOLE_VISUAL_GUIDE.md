```
Mobile (< 640px):
  - Console full width
  - Stats stack vertically below
  - Buttons stack horizontally
  - Font sizes reduced

Tablet (640px - 1024px):
  - Console 100% width
  - Stats in 2-column grid below

Desktop (> 1024px):
  - Console 75% width (left)
  - Stats 25% width sidebar (right)
  - Full feature set visible
  - All buttons side-by-side
```
