**Calculation:**
```
cpuDelta = current - previous (nanoseconds of CPU time)
systemDelta = current - previous (nanoseconds of system time)
cores = number of CPU cores
percentage = (cpuDelta / systemDelta) * cores * 100
Display: "12.5%"
```

**Examples:**
- Idle: 0-5% CPU
- Players active: 10-30% CPU
- Heavy mining/building: 40-80% CPU
- Extreme load: 80-100% CPU
